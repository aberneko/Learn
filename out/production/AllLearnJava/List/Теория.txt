     # add(E e) — Чтобы добавить значения в список, нужен метод. Параметр E e означает, что метод принимает одно
       значение любого из типов. То есть можно передать строку — add("Пиксель"), число — add(17)
       (упаковка примитива в обёртку произойдёт автоматически) или любой объект — add(hamster).
       Метод add(E e) вызывают с помощью имени списка-объекта и точечной нотации.

     # add(int index, E e) Если нужно добавить значение на конкретную позицию — в начало или середину списка.
       У него есть второй параметр — index. Индексация в списках так же, как и в массивах, начинается с нуля.

     (!)    # Если по этому индексу уже хранится какой-то элемент — он станет следующим, а новый займёт его позицию.
            IndexOutOfBoundsException — ошибка из-за выхода за границы списка.

     (!)    # Автоматическое увеличение: Если вы добавляете элемент, а список уже заполнен (т.е. size == capacity),
              ArrayList автоматически создает новый, более крупный внутренний массив (обычно на 50% больше текущего)
              и копирует туда все старые элементы. Это происходит "под капотом" и делает список динамическим.

            # Ручное уменьшение: Вы можете вызвать метод trimToSize(). Этот метод заставит список уменьшить свою емкость,
              чтобы она точно соответствовала текущему размеру (capacity = size). Это полезно для экономии памяти,
              когда вы уверены, что больше не будете добавлять элементы.

            # ArrayList<>(10) — Начальная Емкость (Initial Capacity).
            # Размер (Size) — Фактическое количество элементов, которые в данный момент находятся в списке.

                # Начальная Емкость — Java выделяет память для внутреннего массива, способного хранить 10 ссылок.
                  Все эти 10 ячеек изначально содержат null (пустые ссылки).

     # Вывод списка через "System.out.println()", () — в качестве аргумента может принимать любое значение.

     # get(int index) (от англ. get — «получить») — Чтобы получить элемент из списка, нужно воспользоваться методом.
       В качестве аргумента передаётся индекс.

     # size - Узнать количество элементов. Размер списка равен количеству сохранённых в него значений

     # remove(int index) - Удалить элемент из списка

     # remove(Object o) - он принимает в качестве аргумента любой объект - при передаче примитивов произойдет упаковка
       в класс-обёртку

     # clear() - удаляет все элементы в списке

     # isEmpty() - проверяет есть ли в списке элементы, если есть выводит - false, если нету - true.

     # contains(E e) - Проверить, добавлен ли элемент в список, в качестве аргумента передаётся элемент для поиска.
       Возвращает true или false.
