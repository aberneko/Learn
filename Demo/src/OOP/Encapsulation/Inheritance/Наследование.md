> ### _Позволяет создавать новые классы на основе уже существующих_
> + #### Благодаря наследованию дочерние классы автоматически приобретают функционал класса-родителя. Не нужно раз за разом прописывать одни и те же поля и методы — их можно передать по наследству.
---

> ### _Для передачи полей одного класса к другому требуется ключевое слово ```extends```_
> ```
> public class Animal {}
> public class Dog extends Animal {}
> ```
> 
> + #### Важно запомнить, что у классов в Java может быть сколько угодно предков, но только один родитель. То есть при помощи ключевого слова extends можно наследовать только от одного класса. Написать class Fox extends Animal, Canidae не получится.
> + #### При внесении изменений в родительский класс они автоматически перейдут по наследству.
> ```
> public class Practicum {
>    public static void main(String[] args) {
>        Ellipse Ellipse = new Ellipse();
>        Ellipse.drawCircle();
>        Ellipse.drawSquare();
>        Ellipse.drawRectangle();
>        Ellipse.drawCircle();
>        Ellipse.circleToEllipse();
>    }
>}
>
> class Figure {
>   public void drawCircle() {
>       System.out.println("Рисуем круг");
>   }
>    public void drawSquare() {
>        System.out.println("Рисуем квадрат");
>    }
>    public void drawRectangle() {
>        System.out.println("Рисуем прямоугольник");
>    }
> }
>
> class Circle extends  Figure{
> }
>
> class Square   extends Circle {
> }
>
> class Rectangle extends Square {
> }
>
> class Ellipse extends Square{
>   public void circleToEllipse() {
>       System.out.println("Превращаем круг в овал");
>   }
> }
> ```
 ---
> #### _Класс-родитель, или класс-предок, иначе ещё называют суперклассом. Классы-потомки — подклассами._
